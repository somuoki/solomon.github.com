Urembo Hub Africa

The client

A successfull Kenyan business lady who has been in the beauty industry for a long time. She saw a gap in the market and siezed it without a second thought.

Task

Create a multivendor beauty services and products ecommerce system

urembohub.africa

Urembo Hub Africa and What it entails

Urembo Hub Africa is an e-commerce mobile/web-based digital platform that seeks to bridge the gap between the consumers of beauty products and link beauty professionals, merchants, and Enterprises to sell products and services across Kenya/Africa.

Challenges and Requirements

The client had already acquired several customers who needed to use the system as soon as it was developed. This meant creating a prototype system that would be used by the existing customers while still developing the main fully featured system.

My Aproach to the Project

Agile Prototyping, given that the system needed to be live during development this seemed like the best strategy. Using docker containerisation provided a couple of advantages especially with version control and continous deployment and testing.

Technologies Involved
  • Website Backend. For this system I used (Wordpress) as my framework for faster development, MySQL as my database system and an Apache server.
  • Website Frontend. HTML5, CSS3 and JavaScript (Ajax cart to create a smooth shopping experience)
  • Payment Integration. Paypal, Stripe, Cash on Delivery, Flutterwave(Mainly for mobile money).
  • Future Updates and Features. Mobile Application,
